Simple SF Energy Sharing ABM

Simulation time is explicitly 2025-01-01 00:00 through 2025-01-30 23:00, with no-sun disturbance on 2025-01-15.

Agent state: A_i(t) = {building_type, norm, generation_i(t), demand_i(t), storage_i(t), health_i(t)}. This version uses only residential buildings.

Load profile source: local SF residential rows from energy_profiles_hourly_used.csv, joined to residential buildings in building_energy_metadata.csv by profile_id. The GitHub repo stores the compact prepared version in data/agents_initial.json.

Solar generation potential source: cached NASA POWER 2025 hourly ALLSKY_SFC_SW_DWN for San Francisco. Each hourly value is converted from W/m2 to kWh/m2 by dividing by 1000. Generation uses the same fixed PV area and the same global scaling factor for every grid cell: generation_i(t) = solar_kwh_per_m2(t) * 1.0 m2 * 5.0.

Battery/storage size is fixed for every building: battery_capacity_i = 5.0 kWh. storage_i(t) carries unused surplus energy forward and is capped at 5.0 kWh.

Building health is continuous: health_i(t) = clip((generation + starting_storage + energy_received - energy_exported) / demand, 0, 1). A building is dead at hour t when health_i(t) = 0, and is critical when 0 < health_i(t) < 0.05. If a building stays dead for 24 consecutive hours it is permanently dead: from then on its health is fixed at 0 and it never revives, generates, or shares.

Energy sharing rule: gift = min(surplus, energy_request) * norm_donor. The two scenarios set every residential building's norm_i to 0 or 1.

Performance metrics: % building alive (health > 0), % critical (health < 5%), % permanently dead, and resilience. Here Q(t) is system performance, defined as the average building health: Q(t) = mean_i health_i(t). Resilience is normalized area under that curve: R = integral Q(t) dt / integral Q0 dt, where Q0 = 1.
