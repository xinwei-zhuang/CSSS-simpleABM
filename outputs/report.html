Simple SF Energy Sharing ABM

Simulation time is explicitly 2025-01-01 00:00 through 2025-01-30 23:00, with no-sun disturbance on 2025-01-15.

Agent state: A_i(t) = {building_type, norm, generation_i(t), demand_i(t), storage_i(t), health_i(t)}. This version uses only residential buildings.

Building health is continuous: health_i(t) = clip((generation + starting_storage + energy_received - energy_exported) / demand, 0, 1). A building is dead at hour t when health_i(t) = 0. If a building stays dead for 24 consecutive hours it is permanently dead: from then on its health is fixed at 0 and it never revives, generates, or shares.

Energy sharing rule (one simultaneous step): each building with spare energy splits norm * surplus equally among the neighbors that are short this hour, but never gives a neighbor more than it needs — each gift is min(share, neighbor_shortfall). The two scenarios set every building's norm_i to 0 (share nothing) or 1 (share all spare energy).

Performance metrics: % building alive (health > 0) and resilience. Here Q(t) = mean_i health_i(t) is system performance; resilience is the normalized area under that curve: R = integral Q(t) dt / integral Q0 dt, where Q0 = 1.
