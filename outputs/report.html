Simple SF Energy Sharing ABM

Simulation time is explicitly 2025-01-01 00:00 through 2025-01-30 23:00, with no-sun disturbance on 2025-01-15.

Agent state: A_i(t) = {health_i(t), generation_i(t), demand_i(t), storage_i(t), generosity}.

Building health is continuous: health_i(t) = clip((generation + starting_storage + energy_received - energy_exported) / demand, 0, 1). A building is dead only when health_i(t) = 0.

Energy sharing rule: gift = min(surplus, energy_request) * generosity.

Only two performance metrics are reported: % building alive and resilience. Here Q(t) is system performance, defined as the average building health: Q(t) = mean_i health_i(t). Resilience is normalized area under that curve: R = integral Q(t) dt / integral Q0 dt, where Q0 = 1.
